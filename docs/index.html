MailGUI233

A tiny desktop mail GUI with multi-provider accounts, a mail timetable, editable server presets, cloud sync strategies, and user-owned CSS themes.

Two-column navigation

Accounts stay in the first rail. Mailbox functions stay in the second rail. The layout stays predictable when users switch contexts.

Mail timetable

Day, week, and month views turn mail into scheduled work. Items support create, edit, duplicate, delete, and right-click menus.

Mailbox settings

Provider setup is app-owned UI. IMAP, POP3, and SMTP presets are visible and editable instead of hidden behind native prompts.

Cloud sync and themes

Git, WebDAV, and local-folder sync are strategy-driven. A single custom CSS URL can override the interface as the highest-priority theme layer.

Small package

Windows NSIS installer stays around two megabytes in current builds.

Visual QA

Playwright screenshots verify desktop and mobile product flows.

Agent-first

Install, launch, status, uninstall, package, and verification scripts are designed for automation.
